# OpsLens — Diagnostics → Insights → Trends

Upload a `.tar.gz` generated by `collector/cluster_diag.sh`. OpsLens parses the artifact, stores metrics in SQLite, computes a health score, and shows trends + run diffs in a web dashboard.

## Local run (no Docker)
```bash
cd server
python -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt
uvicorn app.main:app --reload
